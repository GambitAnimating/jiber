Rejibe is a multi-user state container for JavaScript apps.

### Perks
- Offline First
- Optimistic
- Fast (updates are sent via WebSockets)
- Tiny (rejibe-client is 3kb, and has no dependencies)

### Installation
Client
```
npm i rejibe-client
```

Server
```
npm i rejibe-server
```
